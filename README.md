<h2>Leveled Mobs & HUD Display</h2>  
Mobs grow increasingly powerful as distance and days progress! With every level up, they gain random attribute points in Attack Damage, Movement Speed, Defense, Armor Penetration, and Max Health.<br> 
We’ve also included a built-in real-time Health and Damage indicator. If you prefer a cleaner UI, you can easily toggle it off in the configuration menu.  

![攻击怪物1](https://github.com/user-attachments/assets/6e03217a-e268-49e1-a1f8-7e8aab13ac7b)  

<h2>Boss System</h2>  
Monsters have a small chance to spawn as powerful Bosses! Their ranks are categorized as Elite, Lord, and Hero. The higher the rank, the more lethal special skills they possess. Defeating them grants you Legendary Loot!<br>
You can freely adjust the Boss Spawn Rate within the configuration menu.

<h2>Admin Controls</h2>

Requirement: Creative Mode + lmll_admin tag.（/tag @s add lmll_admin）

How to Open: Right-click with a Book in hand.

Function: Fully customize world settings, mob scaling, and system mechanics!

Other Commands

The following commands are only available to players in Creative Mode who possess the lmll_admin tag:

!lmll [Skill Name] [Tier] — Applies the specified skill to the item currently held in your hand.

!setdmg [Value] — Sets the bonus damage for the held item.

!sethp [Value] — Sets the health bonus for the held item.

!setmelee [Value] — Sets the melee defense for the held item.

!setranged [Value] — Sets the ranged defense for the held item.

!setpen [Value] — Sets the armor penetration for the held item.

!setspeed [Value] — Sets the speed for the held item.

!setcrit [Value] — Sets the critical hit rate for the held item.

!setdr [Value] — Sets the damage reduction (DR) rate for the held item.

!punch add — Gives yourself a Socket Punch.

Scoreboard Integration
Additionally, you can adjust a player's Honor or Power by modifying the following scoreboards:

lmll_honor

lmll_power

To track or display these values, use the following detection scoreboards:

lmll_honor_display

lmll_power_display
